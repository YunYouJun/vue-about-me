# vue-about-me

about-me

Display some links about me.

## How To Use

- icon (fontawesome)
- element-ui

You can see `example` folder.

## Options

Attribute | Description | Type | Options | Default
---|---|---|---|---
copyright | 显示版权 | Object{projectName:'',author:'',logo:''} | —— |
links | 显示链接 | Array(Object) | —— |

```js
links: [
  {
    name,
    color,
    icon,
    label,
    href,
  }
]
```
